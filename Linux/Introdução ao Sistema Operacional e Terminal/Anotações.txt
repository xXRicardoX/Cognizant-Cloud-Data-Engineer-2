Comandos basicos terminal linux

Abrir o terminal-> CTRL +ALT+T

pwd -> mostra qual pasta esta no terminal

ls -> lista os diretorios

ls -l -> lista os diretorios com detalhes gerais

ls --help -> mostra as funções do comando ls

mkdir -> criar pastar pelo terminal

cd .. retorna para o diretorio anterior
 
cd / retorna para o diretorio raiz

man ls -> lista o manual (Para sair presione Q)

history -> mostra os comandos digitado no terminal

ctrl + C -> Cancela o comando atual em funcionamento.

Ctrl + Z -> Pausa o comando atual, em primeiro ou segundo plano

Ctrl + D -> Faz o logout da sessão atual

Ctrl+ W -> Apaga uma palavra na linha atual

Ctrl + U -> Apaga uma linha inteira

Ctrl + R -> Busca um comando recente

!! -> Repete o último comando

exit -> Faz o logout da sessão atual

man -> Manual

mv -> mover(mv nome do arquivo ~ um arquivo / renomear (ex: mv teste linux) um diretorio

touch -> cria um arquivo

CP -> copia (Ex: cp linux.txt /home/ricardo) do arquivo

RM -> excluir arquivo

RMDIR -> excluir diretorio
